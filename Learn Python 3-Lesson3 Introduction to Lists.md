4.	Which of the following is the correct way to turn the following into a list of names: “Tom”, “Jerry”, “Tweetie”, “Sylvester”?
-	names = [“Tom” “Jerry” “Tweetie” “Sylvester”]
-	names = “Tom”, “Jerry”, “Tweetie”, “Sylvester”
-	names = {“Tom”, “Jerry”, “Tweetie”, “Sylvester”}
-	**names = [“Tom”, “Jerry”, “Tweetie”, “Sylvester”]**
